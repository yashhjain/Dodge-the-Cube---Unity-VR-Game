# Dodge-the-Cube---Unity-VR-Game
• Designed a Unity VR game where the first-person player dodges and destroys the cubes on an endless path 
• Implemented the VR interactions to destroy and to manoeuvre around the cube using C# scripts

Game Controls:
• Move Player using Left/ Right arrow keys
• Jump using Space bar
• Fire the cubes by left mouse click

Steps to run the project:
• Open Unity Hub
• Uncompress the zip, you will find the Cube Mania – Final folder which you can open in
Unity
• Inside Assets you will find the scene named MainMenu which you can run to play the
game
